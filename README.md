# Sistema Avanzado de Gestión de Inventario

## Descripción
Este proyecto es un sistema avanzado de gestión de inventarios para una tienda, desarrollado en Python usando **Programación Orientada a Objetos (POO)**. Permite gestionar productos de manera eficiente utilizando **colecciones** y almacena la información de forma persistente en archivos JSON.

---

## Estructura del proyecto


Sistema-Inventario-Avanzado/
│
├─ producto.py # Clase Producto: atributos y métodos de cada producto
├─ inventario.py # Clase Inventario: manejo del inventario y persistencia
├─ main.py # Menú interactivo para usar el sistema
├─ inventario.txt # Archivo generado automáticamente para almacenar los productos
└─ README.md # Documentación del proyecto

## Funcionalidades
- Añadir producto
- Eliminar producto por ID
- Actualizar cantidad o precio
- Buscar productos por nombre
- Mostrar todos los productos
- Guardar y cargar automáticamente el inventario

## Cómo usar
1. Asegúrate de tener Python 3 instalado.
2. Abre la terminal en la carpeta del proyecto.
3. Ejecuta el sistema con:
```bash
python main.py
